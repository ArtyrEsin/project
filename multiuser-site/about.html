О платформе

Этот сайт представляет собой универсальный каталог проектов. Он подходит для команды, учебной группы, курса, кружка или небольшого сообщества.

Что умеет сайт

Пользователи могут добавлять проекты, указывать автора, категорию, ссылку, описание, менять статус выполнения и смотреть статистику.

Для кого подходит

Платформа подойдёт для демонстрации работ нескольких людей, а не только одного владельца сайта. Например, для команды разработчиков или группы студентов.

Как пользоваться

  • На главной странице отображаются все проекты из локального хранилища.
  • Через страницу добавления можно создать запись для любого автора.
  • Проекты можно фильтровать по названию, автору и статусу.
  • На странице статистики отображается общая сводка по каталогу.
